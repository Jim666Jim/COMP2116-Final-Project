## Purpose of the Software

### Development Process Applied
**Waterfall Model**

### Why This Process?
As a solo developer with clearly defined requirements, the Waterfall model is appropriate:
- Requirements are fixed (classic Snake mechanics)
- Design is straightforward (grid-based movement)
- Testing can be done after implementation
- No need for iterative customer feedback

### Target Market / Possible Usage
- Casual gamers looking for quick entertainment
- Students learning Python game development
- Retro gaming enthusiasts
- Educational purposes (demonstrating game loop concepts)
